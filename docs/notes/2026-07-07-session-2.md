## What happened

- **Verified TASK-001**: `pip install -e .[test]` + `pytest` in the eros venv —
  14 passed. RUN-001 review upgraded to approve; the REVISE verdict had been an
  artifact of the headless session's permission policy, not of the code.
- **TASK-002 — token bucket**: lock-guarded lazy refill, clamped tokens,
  `available` property, 6 unit tests.
- **TASK-003 — property suite**: `GUARANTEES` metadata in `ratezoo/__init__.py`,
  hypothesis schedules, O(n) sliding-scan window-grant bound, idle-recovery and
  never-raises invariants, seed-driven 2000-op float-drift regression.
  **Spec deviation (documented in the module docstring):** the planned bound
  `capacity * max_burst_factor` alone is unsatisfiable for any refill-based
  limiter (full bucket + one window of refill legitimately grants ~2x capacity),
  so the implemented bound is `capacity * mbf + rate * W`. Teeth check verified:
  mbf 0.5 fails, 1.0 passes. Also added `reset_after_idle_factor` to GUARANTEES
  because the sliding counter genuinely needs two idle windows to fully reset.
- **TASK-004 — leaky bucket + fixed window**: grid-anchored windows;
  `test_fixed_window_boundary_double_burst` documents the classic 2x flaw.
- **TASK-005 — sliding log + sliding counter**: exact deque log with
  `<=`-pruning (so exactly-W idle clears); two-bucket interpolation counter with
  a test pinning one concrete over-admission vs the exact log.
- **TASK-006 — thread safety**: 8 threads x 5000 calls per algorithm, barrier-
  synced, real clock; 5 consecutive clean runs.
- **TASK-007 — bench harness**: `python -m ratezoo.bench` — median-of-5
  throughput (with `--threads`), deterministic steady/spike/ramp FakeClock
  workloads with full traces, stdlib table, `--json`. Spike traces byte-identical
  across runs.
- **TASK-008 — report generator**: `python -m ratezoo.bench.report` merges
  result JSONs into markdown tables + spike summary; exit 2 on bad input.
  `comparison.md` committed as a sample.

**Final state: 62 tests passed; `eros status` shows rate-limiter-zoo complete, 8/8 tasks.**
